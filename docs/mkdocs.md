# MkDocs <!-- omit in toc -->
```bash
python -m venv venv
source venv/bin/activate
pip --version

pip install mkdocs-material

mkdocs new .
mkdocs serve
```

## Plugins
```bash
pip install mkdocs-minify-plugin
pip install mkdocs-search-plugin
pip install mkdocs-social-plugin
```
